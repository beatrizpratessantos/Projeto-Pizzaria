# Projeto Pizzaria 
Desenvolvido em Java utilizando o padrão de projeto Bridge para simular um sistema de montagem de pizzas.

## Objetivo
Demonstrar a aplicação do padrão Bridge, separando:
- Tamanhos de pizza
- Tipos de borda
- Sabores

### Tamanhos de pizza
- Pequeno
- Tradicional
- Grande

### Bordas
- Tradicional
- Catupiry
- Chocolate

## Exemplos de Sabores
- Calabresa
- Mussarela
- Pepperoni
- Chocolate
- Morango
- Banana

## Como Executar

Compile:

```bash
javac *.java
```

Execute:

```bash
java Main
```
